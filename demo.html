Full Stack Project

User Management System

A comprehensive web application enabling user registration, login, and profile management with RESTful API integration

Tech Stack

React Node.js Express MongoDB REST API

User Registration

Secure signup with email verification, password hashing using bcrypt, and input validation

Authentication

JWT-based authentication with secure token management and session handling

MongoDB Integration

Mongoose ODM for schema validation, data modeling, and efficient database queries

Form Validation

Client & server-side validation with comprehensive error handling and user feedback

POST /api/auth/register Create new user account
POST /api/auth/login Authenticate user
GET /api/users/:id Get user profile
PUT /api/users/:id Update user profile
DELETE /api/users/:id Delete user account
GET /api/users List all users (admin)
